# id-scraper
## What is this?
A Python script that retrieves IMDB/TVDB/IGDB IDs.

## How do I use it?
Running locally:
- Install python
- Install script dependencies (bs4, tvdbsimple, requests)
- Execute script
Alternative:
- Go to https://repl.it/github/Ser4ph2/id-scraper and click "Run" at the top to execute in browser
